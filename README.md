# printer-camera-script
Script for the camera attached to the VLA 3D printer

Pseudocode:
1) Create virtual environment
2) Install pyorbbecsdk, opencv-python, and numpy
3) Create local folder for captured images
4) Access Depth and Color frames data from camera
5) Synchronize frames
6) Display live frames on screen
7) If spacebar is pressed, capture frame screenshots
8) Store in captured images folder
